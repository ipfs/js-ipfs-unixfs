    • entries(

      Passing offset and/or length to entry.content() will limit the number of files returned from the directory.

      const entries = []

      for await (const entry of dir.content({
        offset: 0,
        length: 5
      })) {
        entries.push(entry)
      }

      // `entries` contains the first 5 files/directories in the directory
